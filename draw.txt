# 论文实验结果可视化需求

## 数据背景

我有 4 个实验结果表格：

```text
spk_2.csv
spk_3.csv
spk_4.csv
spk_5.csv
```

每个表格结构类似：

```text
speaker,sample,sdr_before,sdr_after,delta_sdr,stoi,pesq
```

前100行为原始测试样本。

末尾若干行为统计结果，例如：

```text
SubAvg: Mild Noise (SDR > -5dB)
SubAvg: Moderate Noise (-10dB < SDR <= -5dB)
SubAvg: Severe Noise (SDR <= -10dB)
== TOTAL AVERAGE ==
```

其中：

* delta_sdr = SDR提升量
* stoi = 语音可懂度
* pesq = 语音质量

---

# 总体要求

生成适合毕业论文的高质量图片。

要求：

* dpi=300
* png保存
* 字体统一（Times New Roman）
* 坐标轴字号统一
* 图例无遮挡
* 使用 matplotlib
* 不使用 seaborn 默认风格
* 每张图单独输出
* 图片尺寸适合A4论文排版

建议：

```python
figsize=(8,5)
dpi=300
```

# 图1：不同说话人数下总体性能趋势图

## 目的

展示说话人数增加对系统性能的影响。

## 数据来源

每个文件中的：

```text
== TOTAL AVERAGE ==
```

对应数据。

## 横轴

```text
2
3
4
5
```

表示说话人数。

## 纵轴

绘制三条曲线：

```text
ΔSDR
STOI
PESQ
```

使用双Y轴：

左轴：

```text
ΔSDR
```

右轴：

```text
STOI
PESQ
```

避免量纲差异导致曲线被压缩。

## 图标题

```text
Overall Performance under Different Numbers of Speakers
```
